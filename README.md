# Portfolio Website

A modern, responsive portfolio website showcasing my work as a web designer, front-end developer, and graphic artist.

## Features

- **Interactive Intro Page**: 18th century-inspired letter animation
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Project Showcase**: Interactive carousel with detailed project modals
- **Skills Section**: Categorized display of technical skills and tools
- **Certifications**: Professional achievements with elegant frame design
- **Contact Form**: Easy-to-use contact form with social media links

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with animations and responsive design
- **JavaScript**: Interactive functionality and animations
- **Google Fonts**: Poppins and Dancing Script fonts

## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process or dependencies required - works out of the box!
